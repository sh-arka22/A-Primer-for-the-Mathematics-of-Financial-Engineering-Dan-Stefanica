## 📖 Overview

This vault covers the complete mathematical toolkit for financial engineering — from pre-calculus foundations to Black-Scholes, Greeks, numerical methods, and optimization. Each day builds on the previous, connecting theory to real trading intuition and interview preparation.

## 🚀 How to Use

1. **Open in Obsidian** — Clone this repo and open the folder as an Obsidian vault
2. **Start at the MOC** — Open [FE Math Primer MOC](FE%20Math%20Primer%20MOC.md) for the full roadmap
3. **Follow the daily workflow:**
   - **① Read** — BFS concept map (understand the landscape)
   - **② Derive** — Work through every derivation by hand
   - **③ Code** — Implement in Python or C++
   - **④ Connect** — Follow wikilinks to related vault notes
   - **⑤ Test** — Interview section — practice Q&A out loud
